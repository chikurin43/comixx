# ComixX (Next.js Prototype)

ComixX は、複数ユーザーで漫画の展開を議論・投票し、作画担当が作品化するサービスのプロトタイプです。

## セットアップ

```bash
npm install
npm run dev
```

## ページ

- `/` 初期ページ
- `/login` ログイン/アカウント作成
- `/main` パレット作成・一覧
- `/palette/[paletteId]` パレット詳細（チャット/投稿/閲覧）
- `/mypage` マイページ

## デプロイ

Vercel にそのままデプロイ可能な Next.js 構成です。
